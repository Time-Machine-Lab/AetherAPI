# Aether Console

`aether-console` is the operator-facing shell for AetherAPI. The current scaffold focuses on:

- an Airbnb-inspired warm visual system defined in `DESIGN.md`
- a reusable console layout with sticky header, category pills, side navigation, and status rail
- overview, sign-in, and protected workspace pages prepared for future module delivery

Common scripts:

- `pnpm dev`
- `pnpm test`
- `pnpm build`

Release and deployment notes are documented in `RELEASE.md`.
